# 选品智能化平台 · 产品功能说明

**文档性质：** 当前 Demo 已实现能力说明  
**产品中文名：** 选品智能化平台  
**英文名：** Merchandising Intelligence Platform  

---

## 1. 产品是什么

选品智能化平台面向服饰电商的选品运营与品类团队，把分散的竞品、内部经营、社媒舆情信息，收拢到同一工作台内完成：

**发现问题 → 形成建议 → 创建提案 → 衔接刊登**

当前版本为可交互前端原型，用于验证信息架构、主流程与页面结构；数据为演示用 Mock，不连接真实后端。

---

## 2. 核心价值主张

| 价值点 | 说明 |
|--------|------|
| 三信号融合 | 竞品网站、产品数据（内部）、社媒舆情一并进入机会判断 |
| 从洞察到动作 | 不止看报表，直接产出建议、方案、提案入口 |
| 可追溯 | 内部建议、舆情归因可下钻到指标与明细，降低「黑箱感」 |
| 链路连贯 | 机会池 / 建议 → 选品方案 / 提案 → 自动刊登任务流 |

---

## 3. 功能全景

### 3.1 监控报告

**竞品网站动态**  
监控竞品上新、排名涨跌、下架等变化，支持筛选与加入机会池。

**内部数据动态（内部选品建议引擎）**  
基于内部销售、行为、退货等数据，识别属性优势、覆盖空档与风险冗余。含三个能力页：

- **建议清单（默认）：** 结构化选品建议表，可转追溯、生成方案草案、加入机会池、改状态  
- **属性钻取：** 按颜色/廓形/面料等维度看适配度、销量、退货与覆盖，支持详情抽屉  
- **明细追溯：** 解释单条建议的指标对比、趋势、代表商品、退货与行为明细  

**社媒舆情动态（VOC 数据中心）**  
聚合多平台用户反馈，完成问题总览、明细搜索、整体分析、分类看板、趋势分析与归因，并可联动机会池 / 避雷榜 / 优化与待办建议。

---

### 3.2 分析洞察

**机会池（主入口）**  
三信号融合后的机会列表与看板：KPI、筛选、机会四象限、流转漏斗、选品方案列表；可查看洞察、生成选品方案并提交。

**竞品热榜**  
竞品上新、热榜、排名涨跌视图，标记是否已进入机会池。

**上新趋势**  
周度上新量、品类占比、站点对比与明细表。

**属性分析**  
按颜色、廓形、领型、面料、设计元素查看属性榜单与四象限。

**定价建议**  
对比竞品与 AZ 价格带，给出建议价与调价动作。

---

### 3.3 选品提案

**提案列表**  
查看提案状态（草稿 / 审核中 / 已通过等），支持同步选款、导出等演示操作。

**创建提案**  
对齐现网选款「创建提案」表单结构：基础信息、选款信息（图片与佐证）、规格信息、补充信息；支持保存草稿、提交校验。  
预留从机会池 / 内部建议一键预填的能力（函数已备，入口可后续打通）。

---

### 3.4 自动刊登

**任务列表**  
刊登任务工作台：新增量、待上海/美国审核、推送异常等 KPI；按 AI 状态、审核状态、盘古推送等筛选；任务「处理」进入审核。

**新建任务**  
录入品类、Goods ID、价格、素材（图片 / 工艺备注）、模特选择，保存后进入 AI 生成。

**AI 审核页**  
任务上下文与指标、价格与中英标题描述、商品属性、图片工作区；支持运营审核通过 / 美国审核通过等双人审核演示。

用于演示权限、映射与权重治理界面，当前为中等保真页面。

## 4. 典型使用路径

1. **找机会：** 机会池 / 竞品热榜 / 内部建议清单 / 舆情槽点榜  
2. **看清楚：** 洞察详情、内部明细追溯、舆情归因  
3. **做决策：** 生成选品方案，或创建选款提案  
4. **往后传：** 提案通过后进入自动刊登任务 → AI 审核 → 推送准备  

---

## 5. 当前范围与限制

**已覆盖：** 全菜单可进入；重点链路（机会池、内部引擎、舆情、创建提案、自动刊登三页）为完整可点页面。  

**尚未包含：** 真实账号体系、数据接入、文件实际上传、与选款/盘古系统的正式接口联调。  

**目录注意：** 请使用嵌套目录中的选品智能化平台 Demo，勿与外层「AI 自动刊登」静态页混淆。

---

## 6. 一句话总结

当前平台已能完整演示：用三信号与内部/舆情引擎发现机会与风险，形成可讨论的选品建议与提案，并接到 AI 自动刊登的任务与审核闭环。
